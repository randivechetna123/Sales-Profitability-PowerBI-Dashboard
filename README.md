Retail Sales Analytics Dashboard

Interactive Power BI dashboard developed using real retail sales data collected from a clothing business (2023–2024).

Objectives :
1. Monitor Sales Performance
2. Analyse Product Profitability
3. Track KPIs
4. Improve Business Decision Making
   
Dashboard Features:
1. Sales Overview
2. Product Analysis
3. Profitability Analysis
4. Dynamic Filters
5. KPI Cards
6. Trend Analysis

Tools Used:
1. Power BI
2. Power Query
3. DAX
4. Data Modeling
5. Excel

Key KPIs
1. Total Sales
2. Total Profit
3. Profit %
4. Average Order Value
5. Total Orders
